# CryptoAPI

透過 Maven 建置的 Spring Boot 專案

---

## 功能說明

- 透過 Coindesk API 取得即時匯率資訊  
  >（API 無法使用，使用 mock 資料作為替代）

---

## 實作加分題

1. **AOP 應用**  
   - 印出所有 API 被呼叫紀錄  
   - 記錄呼叫外部 API 的 Request 與 Response Body 日誌

2. **錯誤處理**  
   - API Response 錯誤處理與管理

3. **Docker 支援**  
   - 可在 Docker 容器中執行
